# Survey Analysis Tool

A web-based application that helps collect, analyze, and visualize survey data.

## Features

- User login and registration
- Survey data collection
- Dashboard with dynamic charts
- Data visualization using Chart.js
- MySQL database integration

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Chart.js

## Project Purpose

This project was developed as part of a Bachelor of Science in Information Technology research project to demonstrate how web-based analytics tools can improve decision-making using survey data.
